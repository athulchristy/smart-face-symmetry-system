# AI Face Symmetry Analyzer

A computer vision based project that analyzes facial symmetry using Python, OpenCV, and dlib facial landmark detection techniques. The system detects important facial points and evaluates facial symmetry by comparing both sides of the face.

## Features

- Facial landmark detection using dlib
- Facial symmetry analysis
- Real-time facial feature visualization
- Symmetry percentage calculation
- Image processing using OpenCV
- Simple and user-friendly implementation

## Technologies Used

- Python
- OpenCV
- dlib
- NumPy
